1.Configure your Git with your name and email. ✍️ git config --global user.name "username" git config --global user.email "gmail" 2.Clone the project repository from GitHub. 🔄 git clone https://github.com/Yashas-DR/git-assignment.git 3.Create a new feature branch. 🌿 git checkout -b new-feature 4.Add a new file called feature.py to your branch. 📂 created a new python file called feature.py git add feature.py 5.Commit your changes with a message. 💬 git commit -m "Add new feature implementation" 6.Push your changes to the remote repository. 🚀 git push origin new-feature 7.Pull the latest changes from the main branch. ⬇️ git pull origin main 8.Remove the feature.py file from your branch. ❌ git rm feature.py 9.Commit the removal of the file and push the changes. 📤 git commit -m "Remove feature.py file" git push origin new-feature
